Sistemas para negocios

Simplifica procesos con soluciones WebApp

Mixa Artesanías — E-commerce Artesanal

Plataforma de comercio electrónico para artesanos mexicanos con catálogo de productos, carrito de compras, gestión de pedidos y pagos, sistema de reseñas y autenticación de usuarios. Desarrollado con Spring Boot, Bootstrap 5 y JavaScript modular.

HM Tracker — App de Entrenamiento (Desktop)

Aplicación de escritorio vibecodeada en Electron + React para llevar un plan de medio maratón de 30 semanas. Panel izquierdo: calendario semanal con fases de color. Centro: rutina de carrera, ejercicio y comidas del día con macros. Derecha: lista de compras autogenerada para la semana.

HM Tracker — Empaquetado como APK para Android

La misma app fue exportada a Android con Capacitor y empaquetada como APK usando Android Studio, e instalada directamente en el teléfono. Así el plan de entrenamiento, dieta y lista de compras viaja en el bolsillo — sin tienda de aplicaciones ni suscripción.
